## Installation

### with gradlew

```bash
./gradlew clean build
java -jar threedollar-api/build/libs/threedollar-api.jar 
```

### with docker-compose

```bash
docker-compose up --build
```

## Tech Stacks
### Language & Framework
- Java 11, Kotlin 1.5
- Spring Boot 2.3 + Spring MVC
- Spring Data JPA + QueryDSL 4.3
- flyway 6.4
- Gradle 7.0
